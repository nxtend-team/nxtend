Runs a schematic that initializes the `@nxtend/capacitor` plugin.

This schematic adds the following Capacitor dependencies:

```
@capacitor/core
@capacitor/cli
@capacitor/android
@capacitor/electron
@capacitor/ios
```

## Usage

```
nx generate @nxtend/capacitor:init
```
